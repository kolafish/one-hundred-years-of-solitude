# 《百年孤独》统计页面

这个 GitHub Pages 站点包含四个独立页面：

- `people/`：人物章节热力图。
- `weekday-events/`：从 `One Hundred Years of Solitude.epub` 中整理 Sunday 到 Saturday 的 117 次星期词提及，并按章节、星期和主题展示对应事件摘要。
- `plants/`：基于英文 EPUB 整理 69 组植物词条，并按类别和章节展示提及次数。
- `animals/`：基于英文 EPUB 整理 75 组动物词条，并按类别和章节展示提及次数。

根目录 `index.html` 是入口页。
